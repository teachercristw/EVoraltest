📄 Google Drive Setup Guide

Follow these exact steps to connect this game to your Google Drive.

  1. Go to Google Drive and create a specific folder where you want to save your results.
  2. Copy the Folder ID from the URL bar (the text after folders/).
  3. Create a new Google Sheet inside that folder.
  4. Click Extensions > Apps Script.
  5. Delete code, paste code below, replace YOUR_FOLDER_ID_HERE with actual Folder ID.
  1. Click Deploy > New deployment. Type: Web app.
  2. Execute as: Me. Who has access: Anyone.
  3. Click Deploy. Authorize permissions. Copy the Web app URL into the Dashboard.
